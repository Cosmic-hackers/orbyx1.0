# Abstract

The ORBYX Tracking System is an advanced, high-fidelity satellite trajectory visualization platform designed for space enthusiasts, amateur astronomers, and mission operators. As low Earth orbit (LEO) becomes increasingly congested due to mega-constellations and scientific deployments, the need for accessible, real-time Space Domain Awareness (SDA) tools is more critical than ever. 

ORBYX bridges the gap between raw orbital telemetry (Two-Line Elements) and user-friendly visual mapping by leveraging modern web frameworks and high-precision Python orbital mechanics libraries. The system automatically fetches actively tracked catalogs from the Celestrak General Perturbations API and renders the data onto three primary interactive modules: an animated 2D Mercator map, an interactive 3D orthographic globe, and a topocentric Sky-View radar oriented to the user's specific ground station coordinates. 

Beyond standard tracking, ORBYX implements an automated alert algorithm for AOS (Acquisition of Signal) and LOS (Loss of Signal) pass predictions, all presented within an immersive, cyberpunk-inspired Mission Control dashboard. The platform also integrates an embedded interactive AI assistant, "Sydh AI," to guide users through its tactical interfaces.
