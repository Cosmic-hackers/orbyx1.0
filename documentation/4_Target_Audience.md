# 4. Target Audience

## 4.1 Primary Audience
The core users of ORBYX include space enthusiasts, amateur radio operators (HAMs), and astrophotographers. These users need incredibly precise times and overhead viewing angles to successfully communicate with satellites (like the ISS) via radio, or to visually track them moving across the night sky. The Sky-View Radar and precise AOS/LOS predictions directly serve this primary demographic.

## 4.2 Secondary Audience
The secondary audience includes students, academic researchers studying orbital patterns, and software developers interested in WebGIS or aerospace data integration. Its visually striking UI also appeals to casual users interested in the aesthetic thrill of a tactical Space Command dashboard.
